# Calculadora de Juros e Descontos
> Projeto DevMedia | Missão 2 | Construa uma aplicação que calcula juros e descontos

Este é um projeto prático desenvolvido em **Portugol** como parte da trilha de **Lógica e JavaScript** da **DevMedia**. O objetivo principal foi aplicar conceitos fundamentais de entrada, processamento e saída de dados.

## Funcionalidades
O programa solicita ao usuário um valor base e um percentual. Com esses dados, ele realiza os seguintes cálculos:
- Calcula o valor correspondente à porcentagem informada.
- Exibe o valor total com o **desconto** aplicado.
- Exibe o valor total com os **juros** aplicados.

## Tecnologias Utilizadas
- **Linguagem:** Portugol (Lógica de Programação)
- **Ferramenta:** Portugol Studio ou interpretadores compatíveis.

## Como funciona a lógica?
O programa utiliza operações matemáticas básicas:
1. **Cálculo da porcentagem:** O sistema divide a porcentagem por 100 e multiplica pelo valor original.
2. **Desconto:** `valor - valor_calculado`
3. **Juros:** `valor + valor_calculado`
